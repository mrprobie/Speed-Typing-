# Speed Typing

A real-time typing speed test application that fetches random quotes and measures your typing accuracy and speed.

Here a link to try it out!!: https://speedtyping0.netlify.app
## Overview

Speed Typing is an interactive web application that challenges you to type randomly generated quotes as quickly and accurately as possible. The app provides live feedback on your typing accuracy with a timer to track elapsed time.

## Features

- **Random Quote Generation**: Fetches random quotes from the Quotable API
- **Real-time Feedback**: Displays typing accuracy with visual indicators
  - ✅ Correct characters displayed in green
  - ❌ Incorrect characters displayed in red with underline
- **Timer**: Displays elapsed time in seconds while you type
- **Auto-focus**: Input field auto-focuses on page load for seamless typing
- **Responsive Design**: Works on desktop and mobile devices



## Technologies Used

- **HTML**: Page structure
- **CSS**: Styling and responsive layout
- **JavaScript**: Interactivity and API integration
- **Quotable API**: Source of random quotes (https://api.quotable.io/)
